# v036 読み込み・保存状態の見える化

## 目的

GAS読込、GASバックアップ読込、下書き保存が成功しているかをエディタ上で明確に確認できるようにする。

## 追加した表示

- 現在の編集データ
- 下書き保存状態
- GASバックアップ状態
- 最終操作

## 状態の見方

- 現在の編集データが `GAS`：GASから読み込んだデータを編集中
- 現在の編集データが `Draft`：ブラウザ下書きを読み込んだ状態
- 現在の編集データが `GAS Backup`：GASバックアップから復元した状態
- 下書き保存が `保存済み`：ゲームで確認できる下書きがブラウザに保存済み
- GASバックアップにbackupIdが表示：GAS backupsシートへの保存または読込が成功

## 次の候補

- エディタから各シートへ直接保存するGAS API
- 保存前の自動バックアップ
- 保存対象シートの選択
