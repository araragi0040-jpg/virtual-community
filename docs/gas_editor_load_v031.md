# v031 GAS読み込み＋エディタ連携

v031では、ゲーム画面だけでなく `editor.html` からもGAS APIのデータを読み込めるようにしました。

## 追加機能

- GAS URL設定
- GAS接続テスト
- GASからエディタへデータ読み込み
- 現在のエディタデータとGASデータの件数差分確認
- GASから読み込んだデータをローカル下書きとして編集・ゲーム確認

## 使い方

1. `editor.html` を開く
2. 「GAS URL設定」でWebアプリURLを登録
3. 「GAS接続テスト」で `mode=ping` の応答を確認
4. 「GASから読込」でスプシ由来データをエディタに読み込む
5. 必要に応じて編集する
6. 「下書き保存」→「ゲームで確認」で反映確認する

## 注意

v031は読み込み専用です。エディタで編集した内容は、まだGAS経由でスプレッドシートへ保存されません。

編集内容を残すには以下を使います。

- 下書き保存
- 全データバックアップ
- 各JSON保存
- TSV/CSV出力

## 次ステップ

v032以降で、エディタからGASへ保存するAPIを設計します。最初は全シート保存ではなく、1シートまたはバックアップシート保存から検証する予定です。
