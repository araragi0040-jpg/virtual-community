# v035 GASバックアップ読込

## 目的

v034で保存できるようになったGASバックアップを、エディタへ読み戻せるようにする。

## 追加ボタン

- GASバックアップ読込
- 最新GASバックアップ読込

## 使い方

1. v035の `gas/Code.gs` をApps Scriptへ貼り替える
2. Webアプリを新しいバージョンで再デプロイする
3. `editor.html` を開く
4. 「GASバックアップ一覧」で backupId を確認する
5. 「GASバックアップ読込」で backupId を入力する
6. または「最新GASバックアップ読込」で最新バックアップを読み込む
7. 必要なら「下書き保存」→「ゲームで確認」

## 注意

バックアップ読込はエディタ上のデータを置き換える。読み込む前の編集内容を残したい場合は、先に「GASバックアップ保存」または「全データバックアップ」を行う。
