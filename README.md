# 歩ける語り場 v017

v017では、配置エディタを一段進めて、対象データの **新規追加・複製・削除** に対応しました。

## v017で追加したこと

- `editor.html` に「新規追加」「複製」「削除」ボタンを追加
- 入口・掲示板・看板・メニュー・イベントなどの interactable を新規追加可能
- NPCを新規追加可能
- 隠し要素を新規追加可能
- 選択中データを複製可能
- 選択中データを削除可能
- interactable の「種類」をフォームから変更可能
- NPCの会話ID、掲示板のboardId、看板のsignIdなどを簡易編集できる「リンク/会話ID」欄を追加
- Undo / Redoで追加・複製・削除も戻せるように対応

## 起動方法

```bash
cd virtual-community-4u-v017
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## 配置エディタの使い方

1. `editor.html` を開く
2. マップとレイヤーを選択
3. 既存の対象をクリックして編集
4. 新しく作る場合は「新規追加」
5. 似た対象を増やす場合は「複製」
6. 不要な対象は「削除」
7. 「下書き保存」→「ゲームで確認」で反映確認

## 注意

新規追加した対象は、まず仮データで作られます。
本番用にする場合は、必要に応じて以下を調整してください。

- 表示名
- 種類
- 座標・範囲
- 反応範囲
- リンク/会話ID
