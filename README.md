# 歩ける語り場 v009

語り場4U・くさのね劇場・ブランチカフェ周辺を歩ける、仮想コミュニティマップのv009です。

## v009で修正したこと

- 称号・実績画面をスクロール可能に修正
- 長い実績一覧でも「閉じる」「体験メモを見る」ボタンが画面内に残るように修正
- スマホ版の上部メニューを縦並びから横スクロール式に変更
- 上部ヘッダーを圧縮し、十字キーが画面内に収まりやすいように調整
- モーダル背景をタップして閉じられるように追加

## 起動方法

`index.html`を直接開くとJSON読み込みで止まる可能性があります。簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v009
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 操作

- 移動：十字キー / WASD / 画面下の移動パッド
- 調べる：スペース / Enter / 画面下の「調べる」ボタン
- 曜日切替：Yキー / 上部の曜日切替ボタン
- 体験メモ：Mキー / 上部のメモボタン
- 称号・実績：Rキー / 上部の実績ボタン
- 座標表示：Tab または ` キー / 上部の座標ボタン


## v009 修正

- 上部メニューを画面レイアウトに影響しないオーバーレイ式に変更。
- 称号・実績画面を長文専用モーダルとして表示し、本文だけスクロールするように修正。
- スマホの縦幅が短い場合でも十字キーが画面内に収まりやすいよう、マップサイズと操作UIを再調整。
