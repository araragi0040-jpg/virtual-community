# 歩ける語り場 v026

v026では、配置エディタを **全対象共通エディタ化** し、NPC・看板・掲示板・メニュー・隠し要素・扉・イベントなどで、表示曜日・表示条件・選択肢を編集できるようにしました。

## 起動方法

```bash
cd virtual-community-4u-v026
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## v026で追加したこと

- NPC、看板、掲示板、メニュー、隠し要素、扉、イベント、ブロックで共通編集欄を表示
- 選択中対象そのものの表示曜日・表示条件をフォームで編集可能
- すべての対象に選択肢エディタを表示可能
- 選択肢がない対象にも「＋選択肢」で後から追加可能
- 掲示板・メニュー・入店確認・隠し要素でも `options` を保存可能
- ゲーム側で、掲示板・メニュー・入店確認・隠し要素の独自 `options` を表示可能
- NPCや掲示板などの表示条件にも `visibleWhen` を利用可能

## 選択肢エディタでできること

- 選択肢の追加・削除
- 選択肢の並び替え
- 表示名の編集
- 動作タイプの選択
- 対象ID / URL / 補足メッセージ / className の編集
- 表示曜日の指定
- 表示条件の指定

## 表示条件の例

- 金曜だけ表示
- 日曜だけ表示
- 会話ポイントが3以上で表示
- カラオケを1回以上体験したら表示
- 特定実績を開放したら表示

## 確認手順

1. `editor.html` を開く
2. NPC・看板・掲示板・メニュー・隠し要素・扉など、任意の対象を選択
3. 「共通設定」で対象の表示曜日・表示条件を編集し、「選択肢エディタ」でボタンを編集
4. 「内容を反映」
5. 「下書き保存」
6. 「ゲームで確認」

## 注意

- 通常の曜日条件はチェックボックスで編集できます。
- 複雑な条件だけ「詳細JSON条件」を使います。
- 下書き保存した内容は、このブラウザ内のプレビュー用データとして保持されます。
