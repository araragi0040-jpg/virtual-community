# 歩ける語り場 v037

v037では、v036の読み込み・保存状態パネルに加えて、**エディタの編集内容をGAS経由でスプレッドシート本体へ保存**できるようにしました。

## v037 更新内容

- `editor.html` に「GASシート保存」ボタンを追加
- エディタ上のデータを `maps / interactables / blocks / npcs / dialogues / boards / menus ...` などの各シートへ保存
- 保存前に、現在のスプレッドシート内容を `backups` シートへ自動バックアップ
- `gas/Code.gs` に `mode=saveSheets` を追加
- 保存結果を「読み込み・保存状態」パネルと出力欄に表示

## 確認方法

```bash
python -m http.server 8000
```

```text
http://localhost:8000/editor.html
```

## 確認ポイント

1. v037の `gas/Code.gs` をApps Scriptへ貼り替える
2. Webアプリを新しいバージョンで再デプロイする
3. `editor.html` を開く
4. `GAS接続テスト` を押す
5. 必要に応じて `GASから読込` を押す
6. 何か軽く編集する
7. `GASシート保存` を押す
8. スプシの各シートが更新されるか確認
9. `backups` シートに保存前バックアップが追加されるか確認
10. `GASから読込` で保存後のデータを再取得できるか確認

## 注意

`GASシート保存` は各シートを上書きします。保存前に自動バックアップを作りますが、初回は必ずテスト用スプレッドシートか、バックアップを取った状態で確認してください。
