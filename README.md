# 歩ける語り場 v020

v020では、配置エディタに **グリッド表示・グリッド吸着・レイヤー表示切替** を追加しました。

## v020で追加したこと

- グリッド表示ON/OFF
- グリッド幅の変更
  - 2.5%
  - 5%
  - 10%
- グリッドに吸着ON/OFF
- レイヤー表示切替
  - 通行不可ブロック
  - 入口/掲示板/看板/メニュー
  - NPC
  - 隠し要素
  - マップ端遷移範囲
- ブロック・NPC・判定範囲の編集時に、吸着設定を反映
- v019の足元ヒットボックス判定はそのまま維持

## 起動方法

```bash
cd virtual-community-4u-v020
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## 配置エディタの使い方

1. `editor.html` を開く
2. マップとレイヤーを選択
3. 必要に応じてグリッド表示・吸着をON
4. 対象をドラッグして移動
5. 四角い範囲は角・辺をドラッグしてリサイズ
6. 「下書き保存」→「ゲームで確認」で反映確認

## 補足

- グリッド吸着をONにすると、ドラッグ移動や数値入力後の座標がグリッド幅に丸められます。
- レイヤー表示をOFFにすると、該当レイヤーは画面上に表示されず、クリック選択もされにくくなります。
- v019で追加した通行不可ブロックの足元当たり判定は維持しています。
