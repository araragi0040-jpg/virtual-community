# 歩ける語り場 v025

v025では、配置エディタに **選択肢ごとの表示曜日・表示条件編集** を追加しました。

## 起動方法

```bash
cd virtual-community-4u-v025
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## v025で追加したこと

- 選択肢ごとの表示曜日をフォームで編集可能
- 選択肢ごとの表示条件をフォームで編集可能
- ゲーム側で、曜日・条件に合わない選択肢を非表示にする処理を追加
- サンプルとして金曜限定・日曜限定・体験値条件付きの選択肢を追加
- `dialogue_options.csv` に `visibleDays` / `conditionJson` を追加

## 選択肢エディタでできること

- 選択肢の追加・削除
- 選択肢の並び替え
- 表示名の編集
- 動作タイプの選択
- 対象ID / URL / 補足メッセージ / className の編集
- 表示曜日の指定
- 表示条件の指定

## 表示条件の例

- 金曜だけ表示
- 日曜だけ表示
- 会話ポイントが3以上で表示
- カラオケを1回以上体験したら表示
- 特定実績を開放したら表示

## 確認手順

1. `editor.html` を開く
2. NPCや看板など、会話に紐づく対象を選択
3. 「選択肢エディタ」で表示曜日・表示条件を編集
4. 「内容を反映」
5. 「下書き保存」
6. 「ゲームで確認」

## 注意

- 通常の曜日条件はチェックボックスで編集できます。
- 複雑な条件だけ「詳細JSON条件」を使います。
- 下書き保存した内容は、このブラウザ内のプレビュー用データとして保持されます。
