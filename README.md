# 歩ける語り場 v036

v036では、GAS読込・GASバックアップ読込・下書き保存が成功しているか分かりにくい問題を改善しました。

## v036 更新内容

- `editor.html` に「読み込み・保存状態」パネルを追加
- 現在の編集データの取得元を表示
  - Local
  - GAS
  - Draft
  - GAS Backup
  - Backup File
- 下書き保存の有無と保存時刻を表示
- GASバックアップの保存/読込状態を表示
- 最終操作の成功/失敗を表示
- GAS読込、下書き保存、バックアップ読込、ゲーム確認移動時にステータスを更新

## 確認方法

```bash
python -m http.server 8000
```

```text
http://localhost:8000/editor.html
```

## 確認ポイント

1. `GASから読込` を押す
2. 「読み込み・保存状態」の「現在の編集データ」が `GAS` になるか確認
3. `下書き保存` を押す
4. 「下書き保存」が `保存済み` になるか確認
5. `GASバックアップ保存` または `最新GASバックアップ読込` を押す
6. 「GASバックアップ」にbackupIdや操作結果が表示されるか確認

## 注意

v036でも、エディタの編集内容を各スプレッドシートへ直接上書き保存する機能はまだ未対応です。
現時点でGASへ保存できるのは、`backups` シートへの丸ごとバックアップです。
