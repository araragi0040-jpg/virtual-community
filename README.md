# 歩ける語り場 v018

v018では、配置エディタとゲーム側に **通行不可ブロック・当たり判定レイヤー** を追加しました。

## v018で追加したこと

- `data/maps.json` に `blocks` を追加
- ゲーム側で、ブロックにぶつかると移動できない処理を追加
- 座標ON時に、通行不可ブロックを赤色で表示
- 配置エディタに「通行不可ブロック」レイヤーを追加
- ブロックの新規追加・複製・削除に対応
- ブロックの移動・リサイズに対応
- ブロック編集も Undo / Redo・下書き保存・ゲームで確認に対応
- スプシ移行用に `data/sheet_templates/blocks.csv` を追加

## 起動方法

```bash
cd virtual-community-4u-v018
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## ブロック編集の使い方

1. `editor.html` を開く
2. レイヤーを「通行不可ブロック」に変更
3. 既存ブロックをクリックして選択
4. 枠の中をドラッグして移動
5. 四隅・辺の白いハンドルをドラッグしてリサイズ
6. 「新規追加」でブロック追加
7. 「下書き保存」→「ゲームで確認」で当たり判定を確認

## 注意

ブロックは `maps.json` 内の各マップに保存されます。
現時点では、建物・木・カウンター・テーブルなどをざっくりブロック化しています。
細かい通路幅や家具の当たり判定は、エディタで調整してください。

## v019 修正

- 通行不可ブロックの判定を、プレイヤー座標の1点判定から「足元ヒットボックス判定」に変更しました。
- 座標ON時に、プレイヤーの実際の当たり判定を青い四角で表示します。
- ブロックに接触している場合は、プレイヤーの当たり判定が赤く表示されます。
- 編集直後にプレイヤーがブロック内にいる場合でも、外へ出る方向の移動だけは許可し、閉じ込められないようにしています。
