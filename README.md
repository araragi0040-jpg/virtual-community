# 歩ける語り場 v016

語り場4Uを中心に、くさのね劇場・ブランチカフェへ移動できる仮想コミュニティMAPです。

v016では、配置エディタを実運用に近づけるために、**下書き保存・ゲーム確認・Undo/Redo・簡易バリデーション**を追加しました。

## v016で追加したこと

- 配置エディタに「戻す」「やり直す」を追加
- 配置エディタに「下書き保存」「下書き読込」「下書き削除」を追加
- 「ゲームで確認」ボタンを追加
  - 編集した配置をブラウザ内に保存し、その配置をゲーム画面に一時反映できます。
  - `index.html?preview=1` で確認できます。
- 簡易バリデーションを追加
  - 判定範囲がマップ外にはみ出していないか
  - 反応範囲が0になっていないか
  - 範囲が小さすぎないか
- 四角い判定範囲は、角・辺をクリック＆ドラッグしてサイズ変更できます。
- `data/schema.json` / `data/sheet_templates/` / `docs/spreadsheet_migration_plan.md` は引き続き同梱しています。

## 起動方法

`index.html` を直接開くとJSON読み込みが止まることがあるため、簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v016
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

編集内容を反映したゲーム確認：

```text
http://localhost:8000/index.html?preview=1
```

## 配置エディタの使い方

1. マップを選択
2. レイヤーを選択
   - 入口・掲示板・看板・メニュー
   - NPC
   - 隠し要素
3. マップ上の枠や点をクリック/タップ
4. ドラッグして座標を調整
5. 四角い範囲は白い角・辺ハンドルでサイズ変更
6. 必要に応じて右側フォームで数値を微調整
7. 「下書き保存」でブラウザに保存
8. 「ゲームで確認」で編集内容を一時反映した状態を確認
9. 問題なければ `maps.json保存` などでファイル出力
10. 出力したJSONを `data/` に差し替える

## 注意

「下書き保存」はブラウザの `localStorage` を使った一時保存です。
正式なファイル更新ではありません。正式反映する場合は、`maps.json保存` / `npcs.json保存` / `hidden.json保存` で出力したJSONを `data/` に差し替えてください。

## スプレッドシート移行について

`data/sheet_templates/` 内のCSVは、Googleスプレッドシートに取り込むための下準備です。

今後の流れは以下です。

```text
JSONでUI・体験を固める
↓
CSVテンプレートでデータ項目を整理
↓
Googleスプレッドシートへ移行
↓
GAS APIでアプリに読み込む
↓
配置エディタからスプシへ保存できるようにする
```
