# 歩ける語り場 v002

語り場4Uを中心に、くさのね劇場・ブランチカフェへ歩いて移動できる仮想コミュニティMAPのv002です。

## v002で追加したこと

- 今日の曜日表示を追加
- 一部NPCの曜日別セリフに対応
- 近くで調べられる対象に「！」マーカーを表示
- 街の掲示板を語り場4U付近マップに追加
- 座標確認モードを追加
  - 画面上の「座標OFF/ON」ボタンで切り替え
  - キーボードの `Tab` または `` ` `` でも切り替え
  - 判定エリア、NPC会話範囲、マップ遷移範囲を可視化
- 掲示板・看板・メニュー・扉の反応範囲を少し広めに設定

## 操作方法

- 移動：十字キー / WASD / 画面下の移動ボタン
- 調べる：Enter / Space / 画面右下の「調べる」
- 座標確認：座標OFF/ONボタン / Tab / `
- 閉じる：Escape

## ローカル確認方法

`index.html`を直接開くとJSON読み込みで止まる場合があります。簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v002
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```
