# 歩ける語り場 v011

語り場4U・くさのね劇場・ブランチカフェ周辺を歩ける、仮想コミュニティマップのv011です。

## v011で修正したこと

- 称号・実績画面をスクロール可能に修正
- 長い実績一覧でも「閉じる」「体験メモを見る」ボタンが画面内に残るように修正
- スマホ版の上部メニューを縦並びから横スクロール式に変更
- 上部ヘッダーを圧縮し、十字キーが画面内に収まりやすいように調整
- モーダル背景をタップして閉じられるように追加

## 起動方法

`index.html`を直接開くとJSON読み込みで止まる可能性があります。簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v011
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 操作

- 移動：十字キー / WASD / 画面下の移動パッド
- 調べる：スペース / Enter / 画面下の「調べる」ボタン
- 曜日切替：Yキー / 上部の曜日切替ボタン
- 体験メモ：Mキー / 上部のメモボタン
- 称号・実績：Rキー / 上部の実績ボタン
- 座標表示：Tab または ` キー / 上部の座標ボタン


## v011 修正

- 上部メニューを画面レイアウトに影響しないオーバーレイ式に変更。
- 称号・実績画面を長文専用モーダルとして表示し、本文だけスクロールするように修正。
- スマホの縦幅が短い場合でも十字キーが画面内に収まりやすいよう、マップサイズと操作UIを再調整。


## v011 修正

- スマホで十字キー以外のボタンを押した後に操作不能になる問題への対策として、ボタンのタップ処理を click 依存から pointerup + click フォールバック方式に変更。
- モーダル右上に × ボタンを追加。
- 非表示メニュー・非表示モーダルがタップを遮らないよう pointer-events を明示。


## v011 修正

- `closeActionDrawer` / `toggleActionDrawer` の不足によるReferenceErrorを修正。
- モーダル表示直前に処理が落ちて `modalOpen=true` のまま固まる問題を防止。
- メニュー・メモ・実績・曜日切替・座標切替後も十字キーと調べるボタンが復帰するように調整。
- 外部リンクを開く前にモーダルを閉じるように変更。
