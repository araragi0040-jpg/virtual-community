# 歩ける語り場 v003

語り場4Uを中心に、くさのね劇場・ブランチカフェへ歩いて移動できる仮想コミュニティMAPのv003です。

## v003で修正したこと

- 座標確認モードをONにしても、十字キーや調べるボタンが下に押し出されないように修正
  - 座標テキストを画面内オーバーレイ表示に変更
- 十字キー長押し時に文字選択されにくいように修正
  - `user-select: none`
  - `touch-action: none`
  - 長押しメニュー抑止
  - pointer capture対応
- 背景画像に描き込まれた人物とNPCスプライトが重ならないように修正
  - v003ではNPC画像を重ね描きしない
  - 話しかけられる地点に近づいた時だけ「！」マーカーを表示
- 低めの画面でも操作ボタンが収まりやすいように、縦幅が小さい場合のUIを少しコンパクト化

## 操作方法

- 移動：十字キー / WASD / 画面下の移動ボタン
- 調べる：Enter / Space / 画面右下の「調べる」
- 座標確認：座標OFF/ONボタン / Tab / `
- 閉じる：Escape

## ローカル確認方法

`index.html`を直接開くとJSON読み込みで止まる場合があります。簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v003
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 確認ポイント

- 座標ONでも1画面内に収まるか
- 十字キー長押しで範囲選択されないか
- 近づいた時だけ「！」が出るか
- 背景の人物と重複表示されていないか
- 「！」が出た状態で調べると会話できるか
