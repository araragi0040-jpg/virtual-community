# 歩ける語り場 v004

語り場4Uを中心に、くさのね劇場・ブランチカフェへ移動できる仮想コミュニティMAPのv004デモです。

## v004で追加したこと

- 曜日切替ボタンを追加
- 曜日ごとのNPC会話差し替えを拡張
- 曜日限定スポットを追加
  - 金曜：語り場4U付近の「金曜のにぎわい」
  - 土曜：くさのね劇場付近の「土曜のイベント気配」
  - 日曜：ブランチカフェ付近の「日曜のカフェ時間」
- 店内にも曜日限定スポットを追加
- 掲示板文言の曜日別表示に対応
- メニュー文言の曜日別表示に対応
- interactablesにもvisibleDaysを設定できるように変更

## 操作方法

- 移動：画面下の十字キー / WASD / 矢印キー
- 調べる：調べるボタン / Enter / Space
- 曜日切替：曜日切替ボタン / Yキー
- 座標表示：座標OFF/ONボタン / Tab / ` キー

## ローカル確認

index.htmlを直接開くとJSONの読み込みが止まる場合があります。
簡易サーバーで起動してください。

```bash
cd virtual-community-4u-v004
python -m http.server 8000
```

ブラウザで以下を開きます。

```text
http://localhost:8000
```

## v005 修正

- v004で発生した黒画面不具合を修正しました。
- 原因は、初期マップ設定前に曜日表示処理からrender()が実行されていたことです。
- 起動順を「画像ロード → 初期マップ設定 → 曜日設定 → ローディング非表示」に変更しました。
- currentMap未設定時のガード処理も追加しています。
