# 歩ける語り場 v023

v023では、配置エディタに **紐づく内容編集** を追加しました。

これまでのように NPC・掲示板・入口・隠し要素・通行不可ブロックの座標を調整できるだけでなく、選択中の対象に紐づく以下のデータも簡易編集できます。

- NPC / 看板 / イベントに紐づく会話データ
- 扉に紐づく入店確認テキスト
- 掲示板タイトル・本文・リンクURL
- メニュータイトル・説明・項目JSON
- 隠し要素の本文・再確認文・出現条件JSON

## 起動方法

```bash
cd virtual-community-4u-v023
python -m http.server 8000
```

ゲーム画面：

```text
http://localhost:8000
```

配置エディタ：

```text
http://localhost:8000/editor.html
```

## 配置エディタの使い方

1. `editor.html` を開く
2. マップとレイヤーを選択
3. マップ上の対象を選択
4. 位置・範囲・反応距離を調整
5. 必要に応じて「紐づく内容編集」で会話文・掲示板文・メニュー文を編集
6. 「内容を反映」
7. 「下書き保存」
8. 「ゲームで確認」

## v023で追加したこと

- `dialogues.json` / `boards.json` / `menus.json` / `linkBoards.json` もエディタで読み込み
- 下書き保存に、配置データだけでなく内容データも含めるように変更
- ゲーム確認時に、内容編集の下書きも反映
- `dialogues.json保存` / `boards.json保存` / `menus.json保存` / `linkBoards.json保存` ボタンを追加
- 選択中オブジェクトの紐づく内容JSONをコピー可能に変更

## 注意

- v023の内容編集は、まだ簡易版です。
- 選択肢・曜日別文言・メニュー項目などは、JSON欄で編集します。
- JSON欄の形式が崩れると反映できないため、まずは少しずつ変更して確認してください。
